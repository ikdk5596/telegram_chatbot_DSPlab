# ETRI 음성인식을 적용한 간단한 ChatBot

1. ETRI 음성인식 api 신청
  http://aiopen.etri.re.kr/index.php
  Open Api 사용 신청 후 Key 발급

2. Telegram bot TOKEN 값 받아오기
  https://core.telegram.org/bots/api

3. main.py 에서 ETRI Key 및 Telegram TOKEN 알맞은 위치에 붙여넣기

4. main.py 실행
